JOGO DO NÚMERO SECRETO:
-Crie um jogo onde o usuário vai precisar advinhar o número secreto com as seguintes regras:

1.O jogo continua enquanto o jogador não acertar.
2.Se o jogador digitar um número maior que o número secreto → mostrar “Muito alto!”.
3.Se digitar um número menor → mostrar “Muito baixo!”.
4.Quando acertar → mostrar “Parabéns, você acertou!” e encerrar o jogo.
