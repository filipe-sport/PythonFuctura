Escreva um programa que receba a idade de uma pessoa, diga em qual categoria ela se encontra
e mostre se é possível ele participar de atividades para maiores de idade ou não.
Utilize como parâmetro:

0 a 12 anos → Criança

13 a 17 anos → Adolescente

18 a 59 anos → Adulto

60 anos ou mais → Idoso

Além disso:
Se for criança ou adolescente mostre a mensagem: "Você ainda é menor de idade"
Se for adulto OU idoso, mostre a mensagem adicional:
→ Pode participar das atividades para maiores de idade.
